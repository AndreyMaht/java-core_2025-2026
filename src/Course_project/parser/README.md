# Парсер курсов валют

Проект реализует парсер веб-сайта Центрального банка России (cbr.ru) для автоматизированного сбора данных о курсах валют, сохранения их в Excel-файл, сравнения с предыдущими значениями и уведомления об изменениях.
## Технологии
- Java (версия 11 или выше)
- Jsoup — для парсинга HTML и работы с DOM
- Apache POI — для чтения и записи Excel-файлов (.xlsx)

## Возможности
- Получение актуальных курсов валют с сайта ЦБ РФ
- Парсинг HTML-страницы и извлечение данных из таблицы
- Экспорт данных в Excel-файл (`rates.xlsx`)
- Чтение ранее сохранённых курсов из Excel
- Сравнение текущих курсов с предыдущими
- Вычисление процентного изменения курса
- Выбор валюты для отслеживания (пользователь вводит код валюты или 'all' для всех)
- Вывод изменений в консоль с форматированием
- Автоматическое обновление Excel-файла только при наличии изменений

## Запуск
1. Открыть проект в IntelliJ IDEA или другой IDE.
2. Убедиться, что библиотеки Jsoup и Apache POI подключены (см. раздел "Зависимости").
3. Запустить класс `CurrencyParser`.
4. В консоли ввести код валюты (например, `USD`) или `all` для всех валют.
5. Программа выведет текущие курсы, сравнит с предыдущими и покажет изменения.
