## Currency Parser 💱

Парсер курсов валют на языке Java с графическим интерфейсом, автоматическим анализом изменений и экспортом данных в Excel.

## О проекте

Данное приложение предназначено для автоматизированного получения курсов валют с сайта Центрального Банка РФ, анализа изменений курсов и сохранения данных в Excel-файл.

Программа поддерживает:

- получение актуальных курсов валют;
- поиск конкретной валюты;
- анализ изменения курса;
- вычисление процента изменения;
- сохранение данных в Excel;
- графический интерфейс пользователя (GUI).

## Возможности приложения

1. [x] Парсинг данных с сайта ЦБ РФ
2. [x] Получение всех курсов валют
3. [x] Поиск валюты по коду (USD, EUR и т.д.)
4. [x] Сравнение старых и новых значений
5. [x] Расчёт процентного изменения курса
6. [x] Автоматическое обновление Excel-файла
7. [x] Графический интерфейс на Java Swing
8. [x] Открытие Excel-файла из приложения

## Используемые технологии
| Технология | Назначение                       |
| ---------- | -------------------------------- |
| Java       | Основной язык разработки         |
| Jsoup      | Парсинг HTML-страниц             |
| Apache POI | Работа с Excel                   |
| Java Swing | Создание графического интерфейса |

## 📖 Описание классов

### Класс Currency
Хранит:
- код валюты;
- название валюты;
- текущий курс.

### Класс CurrencyParser
Отвечает за:
- подключение к сайту;
- парсинг HTML;
- анализ изменений курсов;
- сравнение данных.

### Класс ExcelService
Отвечает за:
- запись данных в Excel;
- чтение Excel-файла.

### Класс CurrencyParserGUI - Графический интерфейс приложения.
Содержит:
- поле ввода валюты;
- кнопки управления;
- область вывода результатов.

## Принцип работы

1. Пользователь вводит код валюты или all
2. Программа подключается к сайту ЦБ РФ
3. Выполняется парсинг HTML-страницы
4. Данные сохраняются во внутреннюю структуру
5. Производится сравнение с предыдущими значениями
6. При наличии изменений Excel-файл обновляется
7. Результат выводится пользователю
